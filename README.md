# \# Inventory Manager

# 

# A desktop application built in C# with WinForms that allows users to manage inventory items and create recipes using stored ingredients. All data is saved locally to text files, making it lightweight and easy to run on any computer.

# 

# \## Features

# \- Add, edit, and delete inventory items

# \- Create and manage recipes made from stored ingredients

# \- Create Menus and schedules for planning

# \- Data persistence using SQlite and Dapper

# \- Simple and user-friendly Windows Forms interface

# 

# \## How to Run

# 1\. Clone the repository  

# 2\. Open the project in Visual Studio  

# 3\. Build and run the solution (no database setup required)

# 

# \## Tech Stack

# \- C#  

# \- .NET WinForms  

# \- System.IO / System. Text  
# \- SQL

# 

# \## Future Improvements

# \- Search and filter inventory items  

# \- Export inventory to CSV or Excel  

# \- Switch to SQLite or SQL Server for database support
